The working RTC and LED files for integration into bigger programs are found here. All versions work exactly as intended except for V5. V6 was used on the final rocket launch. V2 is a prototype that only has the data collection and data storage working. V3 is a prototype file that integrates the transceiver with V2. V4 is the final, working variation of V3 that transmits and receives data consistently while also storing it on an on-board SD card. V5 was attempting to use WiFi to connect the telemetry bay to the airbrakes, however it never worked. V6 is the same as V4, except it has LED code to signify different actions that occur by the telemetry bay (powering on, data send, etc.).
