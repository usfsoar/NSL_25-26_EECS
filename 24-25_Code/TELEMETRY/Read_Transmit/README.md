These files are programs that mostly work and test different parts of the telemetry bay when isolated to only that component. They test whether the data collected by each component is accurate, and if it can be transmitted accurately. Specifically, the GPS, barometer, and SD card files are pretty close to the final versions utilized in the final telemetry bay code (some small changes but overall very similar). The IMU file is very similar to the final version, but the velocity metric is not accurate and was changed in the final version.
