The Digixbee files, gps files, and RTC code are mostly functional examples. Note with the RTC code is that the version here may only work by itself and not when integrated into a larger system. An updated version is present in the telemetry files. The ground integration gps file works, however, how well it was integrated into the final product is a bit unknown. The speaker code is not functional. The rest of the files are related to transmitting and receiving data and testing RTOS tasks (working examples for new people). The purpose of this subsection is to experiment with different programs and devices; it is where prototypes are made.
